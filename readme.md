# kubeweekly-scrapper

This project is build for scrapping [kubeweekly.io](https://kubeweekly.io/) (The weekly newsletters for all things Kubernetes).

#### Requirement
---
- NodeJS (v10.14.1)
- npm (6.13.1)
  
#### How to use
---
__Install Dependencies__
```
npm install
```
__Run The APP__
```
npm start
```
